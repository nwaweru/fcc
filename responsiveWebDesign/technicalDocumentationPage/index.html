Getting Started

This is as easy as 1-2-3:

  1. Install composer on your machine. A global installation is recommended so you can follow along. Adjust the commands in the documentation to fit your installation.
  2. Open up your terminal and navigate to your project root.
  3. Run composer require demo/package in your project root to include the myAPI package.

That's it! You are ready to consume myAPI!

Create Aliases

Since I already mentioned it, I will explain. Note: I am using Ubuntu 18.04

  1. nano ~/.bashrc
  2. Navigate to the bottom of the file.
  3. Add an alias entry e.g. for updating my OS.:
    alias update="sudo apt update && sudo apt upgrade && sudo apt dist-upgrade"
    update is just a made up name. I am sure you can do a better codeName.
  4. Save (and, optionally, close) the file.
  5. Run the command source ~/.bashrc
